QZone-Cloud-Like
================

QQ空间云端点赞系统，第一版（测试）

要使用，只需要简单几步：

1.下载代码包并上传到您的网页空间

2.修改Application/config.php，填入您的数据库信息

3.将install.sql导入到您的数据库中

4.用监控宝或者添加Cron任务，地址为   http://您的网站/cron.php ，执行频率可以自己考虑

5.Enjoy it！
